Setup guide

From download to live — A to Z

One-time setup takes about three minutes. After that it's two taps: Start on the phone, Connect on the PC.

Install the Windows app

Download Fonocam-Setup.exe below and run it. If SmartScreen asks, choose More info → Run anyway — the app isn't code-signed yet, and the source is public.

Install the OBS driver (once)

Fonocam uses the OBS Virtual Camera driver to appear as a real webcam. Install OBS Studio from obsproject.com once — you never need to open it.

Install the Android app

Download Fonocam.apk on your phone and install it. Android will ask to allow installs from your browser — allow it, this is normal for apps outside the Play Store.

Press Start on the phone

Open Fonocam, allow the camera and microphone, and tap the big ▶ Start button. The status chip switches to WAITING FOR PC.

Connect from the desktop

Open Fonocam on the PC — your phone is already in the device list (same WiFi). Click Connect. Live preview appears with FPS and resolution.

Start the virtual webcam

Press Start Virtual Webcam. In Zoom, Meet, Discord or OBS, pick the camera named OBS Virtual Camera. You're live.

Optional: go USB

Enable Developer options → USB debugging on the phone, plug in the cable and press USB in the desktop app — lower latency, zero WiFi drops, and the phone charges.

Optional: tune and record

Use the Video and Phone tabs to frame and tune the image, hit ⏺ to record on the PC, or the phone's record button for a 4K backup with sound.

FAQ

Questions, answered

Is Fonocam really free?

Yes. Everything you see today is free and open source — no ads, no account, no watermark. A one-time Pro upgrade with extras like full-HD streaming may come later, but the core app stays free.

Do I need OBS Studio?

Only its virtual camera driver, which installs together with OBS Studio (free). You never have to open or learn OBS — Fonocam just uses the driver so Zoom, Meet and other apps can see your phone as a camera.

Does it work over USB?

Yes. Enable USB debugging on the phone, plug in a cable and press USB in the desktop app. USB is faster and more stable than WiFi — and it charges your phone while streaming.

The video lags on WiFi. What can I do?

Use 720p with Medium quality for the smoothest stream, keep the phone near your router — or switch to a USB cable, which takes WiFi out of the equation entirely.

Does it keep working when I leave the app or the screen dims?

Yes — streaming runs as a background service with a notification. The built-in battery saver dims the screen while streaming continues. Some phone brands pause the camera if the screen turns fully off, so the battery saver is the most reliable mode for long sessions.

Can it record video with sound?

Yes, in two places at once: the desktop records the processed stream on your PC, and the phone records a smooth hardware-encoded backup up to 4K with microphone audio — even if the WiFi lags mid-session.

How do updates work?

The Android app checks GitHub for new releases — Settings → Check for update → one tap to download. For the desktop app, grab the latest installer from the releases page.

Why does Windows say "unknown publisher"?

Fonocam isn't code-signed with a paid certificate, which is normal for free open-source tools. Choose "More info → Run anyway". The full source code is public on GitHub, so you can verify exactly what the app does.
